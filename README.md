# cifar10-cNN-classification-project

This project builds and evaluates a Convolutional Neural Network for image classification using the CIFAR-10 dataset.

## Goals
- Load and preprocess image data
- Visualize images and labels
- Build a custom CNN model
- Train using validation data and early stopping

## Dataset
CIFAR-10: 60,000 color images, 32x32 pixels, 10 classes.
